# Ajuda Bairro

## Integrantes

* Seu Nome

## Tema

Aplicativo para ajudar moradores a registrar problemas urbanos.

## Tecnologias

* React Native
* Expo
* Firebase
* React Navigation

## Funcionalidades

* Criar problemas
* Visualizar problemas
* Editar problemas
* Excluir problemas

## Como rodar

npm install

npx expo start

Depois escaneie o QR Code usando o Expo Go no iPhone.
